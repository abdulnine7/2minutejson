Created using
Express + NeDB.
Deployed on Heroku.

Description:

2MinuteJSON is a free tool to share json data between apps using simple one time password. The API accepts post request on the root url and generates a OTP. The data is stored for 2 minutes on the server DB. Once the 2 minute time is exhausted or the data is retrived using OTP the data is flushed from the server DB.

Routes

Only GET and POST HTTP methods are supported.

Method Route Parameters
GET /json/? otp (number)
POST / json (string)
